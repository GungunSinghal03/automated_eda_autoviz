# 📊 AutoViz Data Analysis Project

This project uses **AutoViz** to automatically visualize and analyze datasets in Python using Jupyter Notebook.

## 🚀 Features
- Automatic data visualization
- Handles large datasets
- Generates graphs and insights quickly
- Works with CSV and structured data

## 🛠️ Tech Stack
- Python
- Pandas
- AutoViz
- Jupyter Notebook

## 📂 File Structure
- `autoviz.ipynb` → Main notebook for analysis
- `data.csv` → Dataset used (if applicable)
- `output.html` → Generated report (optional)
